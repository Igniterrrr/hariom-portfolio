## Overview

**hariom-portfolio** is my personal web portfolio — a modern, visually engaging platform to showcase my skills, achievements, and professional journey.
It’s built to be fast, responsive, and captivating, blending design aesthetics with smooth interactions to create a memorable digital presence.

Through this site, you can explore my **projects, skills, and career highlights** while experiencing the very technologies I use in action.

## Getting Started

### Prerequisites

* Any modern web browser
* Git installed on your system

---

### Installation

```bash
# Clone the repository
git clone https://github.com/Igniterrrr/hariom-portfolio

# Navigate into the project directory
cd hariom-portfolio
```

*(No build tools required — it’s pure HTML/CSS/JS)*

---

### Usage

Simply open the `index.html` file in your browser or serve it locally using any static server.
